# Tibidoh

Tibidoh - Local Mirror Of Internet (TiB Data Over Home).

### Minimal requirements

x86_64/aarch64/riscv64; Debian 13(with tun/tap kernel module); 1 Core; 1GB RAM; 10GB SSD; 4mbps WAN; Open Ports TCP+UDP: 4002  

### Installation

1. Run `git clone --recursive https://github.com/quasin/tibidoh.git tibidoh && cd tibidoh && bash install.sh`
2. System will reboot.
3. Password to NextCloud admin is in ~/.secrets
