# Tibidoh

Dibidoh - Local Mirror Of Internet (TiB Data Over Home).

### Requirements

x86_64/aarch64/riscv64; Debian 13; 2 Core; 4GB RAM; 40GB SSD; 4mbps WAN; Open Ports TCP+UDP: 4002  

### Installation

1. Run `git clone --recursive https://github.com/quasin/tibidoh.git tibidoh && cd tibidoh && bash install.sh`
2. System will reboot.
